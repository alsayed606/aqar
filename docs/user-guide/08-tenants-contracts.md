# 8) المستأجرون والعقود

بعد إضافة عقاراتك ووحداتك، سجّل مستأجريك وأنشئ عقود الإيجار.

## المستأجرون
من الشريط العلوي اضغط **المستأجرون**.

### إضافة مستأجر
1. اكتب **اسم المستأجر** (مطلوب).
2. اختر **النوع**: فرد / شركة.
3. (اختياري) أدخل **رقم الهوية/الإقامة**، **الجوال**، **البريد الإلكتروني**.
4. اضغط **إضافة المستأجر**.

## العقود
من الشريط العلوي اضغط **العقود**.

### إنشاء عقد
> تحتاج إلى وحدة واحدة على الأقل ومستأجر واحد قبل إنشاء عقد.

1. اختر **الوحدة** و**المستأجر**.
2. اختر **نوع العقد**: سكني (بدون ضريبة) أو تجاري (ضريبة 15%).
3. اختر **دورية الدفع**: شهري / ربع سنوي / نصف سنوي / سنوي / دفعة واحدة.
4. حدّد **تاريخي البداية والنهاية**.
5. أدخل **الإيجار السنوي** (بالريال)، و(اختياري) **التأمين** و**رسوم الخدمات**.
6. (اختياري) **رقم العقد** — يُولّد تلقائياً إن تُرك فارغاً.
7. اضغط **إنشاء العقد**. يُنشأ **كمسودة**.

### تفعيل العقد
افتح العقد، ثم اضغط **تفعيل العقد**. عند التفعيل:
- يُولَّد **جدول الاستحقاقات** تلقائياً حسب الدورية (مثلاً: إيجار سنوي 120,000 ر.س ربع سنوي ← 4 استحقاقات بقيمة 30,000 ر.س لكل ربع).
- تُحسب **ضريبة القيمة المضافة** على الإيجار التجاري (15%)؛ الإيجار السكني معفى.
- تصبح حالة الوحدة **«مؤجرة»**.

> ⚠️ **العقد بعد التفعيل مستند قانوني لا يُعدَّل مباشرة.** أي تغيير لاحق سيكون عبر ملحق موثّق (قريباً).

### تسجيل الدفعات ومتابعة الحالة المالية
في صفحة العقد، يظهر **جدول الاستحقاقات** بحالة كل استحقاق:
- **مدفوع** (أخضر) · **متأخر** (أحمر، تجاوز تاريخه ولم يُسدَّد) · **غير مدفوع** (أصفر).

لتسجيل دفعة: أدخل المبلغ (يظهر تلقائياً المتبقّي)، اختر طريقة الدفع، ثم اضغط **دفع**.
- الدفع الجزئي يقلّل المتبقّي.
- الدفع الزائد يُسجَّل رصيداً للمستأجر.

يظهر أعلى الجدول إجمالي **المتبقّي من الإجمالي**.

### ملاحق العقد (تعديل العقد النشط)
العقد بعد التفعيل **ثابت** — أي تغيير يكون بملحق موثّق. من صفحة العقد النشط، قسم **«ملاحق العقد»**:
- **تعديل الإيجار:** أدخل الإيجار السنوي الجديد، تاريخ السريان، والسبب. يُعاد تسعير **الاستحقاقات المستقبلية غير المدفوعة فقط** بالسعر الجديد؛ الماضية والمدفوعة تبقى كما هي.
- **الإنهاء المبكر:** أدخل التاريخ والسبب. يصبح العقد **«مُنهى»**، وتُلغى الاستحقاقات المستقبلية غير المدفوعة، وتعود الوحدة **شاغرة** (جاهزة لعقد جديد).

يظهر **سجلّ الملاحق** (النوع، التغيير، تاريخ السريان، السبب) في نفس الصفحة كأثر موثّق لكل تعديل.

> التمديد حالياً يكون بإنشاء عقد لاحق جديد (حفاظاً على ثبات العقد الأصلي).
