# 5) الميزات القادمة

المنصة تُبنى على مراحل. هذا ما يعمل الآن وما هو قادم. (قد تتغيّر الأولويات.)

## ✅ متاح الآن
- تسجيل الدخول برقم الجوال (رمز تحقق).
- إنشاء منشأة والتبديل بين منشآتك.
- **العقارات والوحدات:** إضافة عقاراتك ووحداتها وحالة كل وحدة.
- **المستأجرون والعقود:** إنشاء العقود، توليد جدول الاستحقاقات، تسجيل الدفعات، والحالة المالية المشتقّة (مدفوع/متأخر/متبقّي).

- **استيراد Excel:** رفع عقاراتك ووحداتك ومستأجريك وعقودك دفعة واحدة، بقوالب عربية جاهزة وتقرير أخطاء لكل صف ومعاينة قبل الاعتماد وإمكانية التراجع.
- **الملّاك وكشوف الحسابات:** ربط العقار بمالك، نسبة أتعاب الإدارة، وكشف حساب المالك لفترة (المُحصَّل − الأتعاب = الصافي).

## 🔜 المرحلة القادمة
- **ملاحق العقود:** تعديل العقود النشطة عبر ملاحق موثّقة.
- **توريد المالك:** تسجيل توريد المبالغ للمالك وكشف حساب قابل للطباعة.

## 🗓️ لاحقاً
- **دعوة الأعضاء** وإدارة الصلاحيات والنطاقات.
- **بوابة المالك:** كشوف حساب دورية للمالك.
- **بوابة المستأجر:** عرض العقود والدفعات للمستأجر.
- **الفوترة الإلكترونية (ZATCA المرحلة الثانية):** فواتير ضريبية نظامية.
- **التحصيل الإلكتروني:** الدفع عبر مدى / Apple Pay مع إيصالات تلقائية.
- **إشعارات واتساب / SMS:** تذكير بالدفعات وتجديد العقود.
- **الشقق المفروشة والحجوزات اليومية:** تقويم حجوزات وتسعير ليلي.
- **تطبيق الجوال.**
