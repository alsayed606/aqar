# 5) الميزات القادمة

المنصة تُبنى على مراحل. هذا ما يعمل الآن وما هو قادم. (قد تتغيّر الأولويات.)

## ✅ متاح الآن
- تسجيل الدخول برقم الجوال (رمز تحقق).
- إنشاء منشأة والتبديل بين منشآتك.
- **لوحة المؤشرات:** نسبة الإشغال، تحصيل الشهر، المتأخرات، إجمالي المستحق، والعقود النشطة/المنتهية قريباً — عند كل دخول.
- **العقارات والوحدات:** إضافة عقاراتك ووحداتها وحالة كل وحدة.
- **المستأجرون والعقود:** إنشاء العقود، توليد جدول الاستحقاقات، تسجيل الدفعات، والحالة المالية المشتقّة (مدفوع/متأخر/متبقّي).

- **استيراد Excel:** رفع عقاراتك ووحداتك ومستأجريك وعقودك دفعة واحدة، بقوالب عربية جاهزة وتقرير أخطاء لكل صف ومعاينة قبل الاعتماد وإمكانية التراجع.
- **الملّاك وكشوف الحسابات:** ربط العقار بمالك، نسبة أتعاب الإدارة، وكشف حساب المالك لفترة (المُحصَّل − الأتعاب = الصافي).
- **سند القبض:** سند مرقّم قابل للطباعة لكل دفعة مستلمة (بالتفقيط) — إثبات التحصيل.
- **الفاتورة الضريبية (ZATCA — المرحلة الأولى):** فاتورة مرقّمة لكل استحقاق، برمز QR وبيانات المورّد/المشتري وتفصيل الضريبة (سكني معفى / تجاري 15%)، قابلة للطباعة.
- **الإشعار الدائن والمدين:** إلغاء فاتورة (إشعار دائن) أو إضافة مبلغ عليها (إشعار مدين)، مرقّم وقابل للطباعة.
- **توريد المالك:** تسجيل ما يُدفع للمالك (سند صرف مرقّم)، وكشف حساب قابل للطباعة (صافي − مورّد = متبقّي).
- **الفريق والصلاحيات:** دعوة موظّفي المكتب برابط انضمام وأدوار، وإدارة أعضائك.
- **نطاقات العقارات للأعضاء:** حصر عضو في عقارات محدّدة (يفرضه النظام تلقائياً).
- **ملاحق العقود:** تعديل الإيجار أو الإنهاء المبكر للعقد النشط عبر ملحق موثّق.
- **بوابة المالك:** دخول ذاتي للمالك لعرض كشوفه وتوريداته وعقاراته.
- **بوابة المستأجر:** دخول ذاتي للمستأجر لعرض عقوده واستحقاقاته ودفعاته.

## 🔜 المرحلة القادمة
- **ربط ZATCA (المرحلة الثانية):** المقاصة/الإبلاغ والتوقيع التشفيري عبر منصة «فاتورة».
- **التحصيل الإلكتروني:** دفع المستأجر عبر البوابة (مدى/Apple Pay).
- **تمديد العقود:** عبر عقد لاحق (يحافظ على ثبات العقد الأصلي).

## 🗓️ لاحقاً
- **دعوة الأعضاء** وإدارة الصلاحيات والنطاقات.
- **بوابة المالك:** كشوف حساب دورية للمالك.
- **بوابة المستأجر:** عرض العقود والدفعات للمستأجر.
- **الفوترة الإلكترونية (ZATCA المرحلة الثانية):** فواتير ضريبية نظامية.
- **التحصيل الإلكتروني:** الدفع عبر مدى / Apple Pay مع إيصالات تلقائية.
- **إشعارات واتساب / SMS:** تذكير بالدفعات وتجديد العقود.
- **الشقق المفروشة والحجوزات اليومية:** تقويم حجوزات وتسعير ليلي.
- **تطبيق الجوال.**
