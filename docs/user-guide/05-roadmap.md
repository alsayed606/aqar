# 5) الميزات القادمة

المنصّة تُبنى على مراحل وفق [ميثاق المشروع](../foundation/07-project-charter.md). هذا ما يعمل الآن وما هو قادم. (الأولويات تُدار عبر الميثاق.)

## ✅ متاح الآن (حيّ)
- **الدخول:** تسجيل الدخول برقم الجوال (رمز تحقق) مع **العودة التلقائية** إلى الصفحة المقصودة بعد الدخول.
- **المنشآت:** إنشاء منشأة والتبديل بين منشآتك.
- **لوحة المؤشرات:** الإشغال، تحصيل الشهر، المتأخرات، إجمالي المستحق، والعقود النشطة/المنتهية قريباً.
- **العقارات والوحدات:** إضافتها وحالة كل وحدة.
- **المستأجرون والعقود:** إنشاء العقود، جدول الاستحقاقات، تسجيل الدفعات، والحالة المالية المشتقّة (مدفوع/متأخر/متبقّي).
- **ملاحق العقود:** تعديل الإيجار أو الإنهاء المبكر للعقد النشط.
- **تجديد العقود:** بعقد لاحق يحافظ على ثبات العقد الأصلي.
- **استيراد Excel:** رفع العقارات والوحدات والمستأجرين والعقود دفعة واحدة، بقوالب عربية وتقرير أخطاء ومعاينة وتراجع.
- **الملّاك وكشوف الحسابات:** ربط العقار بمالك، نسبة الأتعاب، وكشف حساب لفترة (المُحصَّل − الأتعاب = الصافي).
- **توريد المالك:** سند صرف مرقّم + كشف قابل للطباعة.
- **سند القبض:** سند مرقّم قابل للطباعة لكل دفعة (بالتفقيط).
- **الفاتورة الضريبية (ZATCA — المرحلة الأولى):** فاتورة مرقّمة بـ QR وتفصيل الضريبة (سكني معفى / تجاري ١٥٪).
- **الإشعار الدائن والمدين:** إلغاء فاتورة أو إضافة مبلغ عليها، مرقّم وقابل للطباعة.
- **الفريق والصلاحيات والنطاقات:** دعوة الموظّفين بأدوار، وحصر العضو في عقارات محدّدة.
- **بوابة المالك:** دخول ذاتي لعرض الكشوف والتوريدات والعقارات + مستندات قابلة للطباعة.
- **بوابة المستأجر:** دخول ذاتي لعرض العقود والاستحقاقات والدفعات + طباعة سند القبض.

## 🔜 المرحلة القادمة (Beta / v1.0)
- **إشعارات SMS/واتساب:** تذكير بالدفعات وتجديد العقود ([قرار المزوّد: ADR-0001](../adr/0001-sms-provider.md)).
- **تحسينات القوائم:** بحث وفلترة وترقيم صفحات.
- **تفعيل نموذج الاشتراك (SaaS Subscription):** خطط وحدود واشتراك المكتب.
- **تحسينات هندسية (Sprint B):** اختبارات مُلتزَمة + CI، فرض الأدوار الكامل، إزالة الكود المهجور.

## 🗓️ لاحقاً (v2.0)
- **التحصيل الإلكتروني:** دفع المستأجر عبر البوابة (مدى / Apple Pay) مع سند تلقائي.
- **التكامل الحكومي عند توفّر المتطلبات:** ZATCA المرحلة الثانية (المقاصة/الإبلاغ) وربط إيجار (Ejar).
- **طلبات الصيانة** ومزوّدو الخدمة.
- **رفع المستندات والمرفقات.**
- **محرّك تقارير + تصدير PDF/Excel.**
- **الشقق المفروشة والحجوزات اليومية · تطبيق الجوال (PWA أولاً).**
