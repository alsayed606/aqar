# 5) الميزات القادمة

المنصة تُبنى على مراحل. هذا ما يعمل الآن وما هو قادم. (قد تتغيّر الأولويات.)

## ✅ متاح الآن
- تسجيل الدخول برقم الجوال (رمز تحقق).
- إنشاء منشأة والتبديل بين منشآتك.

## 🔜 المرحلة القادمة — سجل الحقيقة (إدارة بياناتك)
- **العقارات والوحدات:** إضافة وتعديل عقاراتك ووحداتك، وحالة كل وحدة.
- **الملّاك والمستأجرون:** سجل كامل لأطرافك.
- **العقود:** إنشاء عقود الإيجار مع جدول الدفعات، والتعديل عبر ملاحق موثّقة.
- **الحالة المالية:** عرض الاستحقاقات والدفعات لكل عقد/وحدة، وحساب المتأخرات تلقائياً.
- **استيراد Excel:** رفع ملف يحتوي عقاراتك ووحداتك وعقودك دفعة واحدة، مع:
  - قوالب جاهزة بالعربية.
  - تقرير أخطاء لكل صف (رقم الصف + الحقل + السبب).
  - معاينة قبل الاعتماد، وإمكانية التراجع عن دفعة كاملة.

## 🗓️ لاحقاً
- **دعوة الأعضاء** وإدارة الصلاحيات والنطاقات.
- **بوابة المالك:** كشوف حساب دورية للمالك.
- **بوابة المستأجر:** عرض العقود والدفعات للمستأجر.
- **الفوترة الإلكترونية (ZATCA المرحلة الثانية):** فواتير ضريبية نظامية.
- **التحصيل الإلكتروني:** الدفع عبر مدى / Apple Pay مع إيصالات تلقائية.
- **إشعارات واتساب / SMS:** تذكير بالدفعات وتجديد العقود.
- **الشقق المفروشة والحجوزات اليومية:** تقويم حجوزات وتسعير ليلي.
- **تطبيق الجوال.**
