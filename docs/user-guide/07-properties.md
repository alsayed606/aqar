# 7) إدارة العقارات والوحدات

بعد الدخول واختيار منشأتك النشطة، يمكنك إدارة عقاراتك ووحداتها.

## عرض العقارات
من الشريط العلوي اضغط **العقارات**. ستظهر كل عقارات منشأتك النشطة، مع نوع كل عقار وموقعه.

> ترى فقط عقارات المنشأة النشطة. للتبديل إلى منشأة أخرى، ارجع إلى **الرئيسية** واختر "التبديل إليها".

## إضافة عقار
في صفحة العقارات، ضمن **"إضافة عقار"**:
1. اكتب **اسم العقار** (مطلوب) — مثال: «برج الياسمين».
2. اختر **النوع**: سكني / تجاري / مختلط / أرض / أخرى.
3. (اختياري) أدخل **رقم الصك**، **المدينة**، **الحي**.
4. اضغط **إضافة العقار**. سيظهر مباشرة في القائمة.

> المالك الافتراضي للعقار هو منشأتك نفسها. لاحقاً ستتمكّن من ربط العقار بمالك محدّد (عميل) وإدارة كشوف حسابه.

## فتح عقار وإدارة وحداته
اضغط على أي عقار لفتح صفحته. ستجد بياناته وقائمة **الوحدات**.

### إضافة وحدة
ضمن **"إضافة وحدة"**:
1. **رقم الوحدة** (مطلوب) — مثال: `101`. لا يمكن تكرار نفس الرقم داخل العقار الواحد.
2. **الحالة**: شاغرة / مؤجرة / محجوزة / تحت الصيانة / غير صالحة للتأجير / خارج الخدمة.
3. (اختياري) **الدور**، **المساحة (م²)**، **غرف النوم**، **دورات المياه**.
4. اضغط **إضافة الوحدة**.

تظهر الوحدات في جدول مع حالة كل وحدة بلون مميّز.

> 💡 تُحفظ **حالة كل وحدة عبر الزمن** تلقائياً، لتُحسب نسبة الإشغال لاحقاً بدقّة.

## الخطوات القادمة
- ربط العقار بمالك (عميل) محدّد.
- إضافة المستأجرين والعقود وجداول الدفعات.
- استيراد العقارات والوحدات دفعة واحدة من Excel.
