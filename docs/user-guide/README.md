## ما المتاح الآن؟

المنصة قيد التطوير التدريجي. المتاح حالياً:

- ✅ **تسجيل الدخول برقم الجوال** (رمز تحقق OTP).
- ✅ **إنشاء منشأة** (مكتب إدارة أملاك / مكتب عقاري) والتبديل بين منشآتك.
- ✅ **إدارة العقارات والوحدات** (إضافة العقارات ووحداتها وحالتها).
- ✅ **المستأجرون والعقود** (إنشاء العقود، توليد الاستحقاقات، تسجيل الدفعات، الحالة المالية).
- ✅ **استيراد Excel** (رفع العقارات/الوحدات/المستأجرين/العقود دفعة واحدة بقوالب جاهزة).
- ✅ **الملّاك وكشوف الحسابات** (ربط العقار بمالك، نسبة الأتعاب، كشف حساب المالك لفترة).
- ✅ **لوحة المؤشرات** (الإشغال، تحصيل الشهر، المتأخرات، إجمالي المستحق، العقود النشطة والمنتهية قريباً).

قيد التطوير (قريباً):

- ⏳ الفاتورة الضريبية النظامية (ZATCA) والتحصيل الإلكتروني.
- ⏳ ملاحق تعديل العقود، وبوابتا المالك والمستأجر.
- ⏳ توريد مبالغ المالك وكشف حساب قابل للطباعة.

راجع [الميزات القادمة](05-roadmap.md) للتفاصيل.
