# ٢) مبادئ المنتج — Product Principles
*عقار (Aqar) · وثيقة تأسيسية ٢ من ٧ · مُعتمدة*

> **الاستخدام:** بوابة القبول/الرفض لكل ميزة مستقبلية. ما يخالف مبدأً جوهرياً يُرفض أو يُعاد تصميمه. تُستشهد بالأرقام (مثال: «يخالف م-٣»).

## أولاً: فلسفة المنتج
في نظام يمسك مال الناس وعقودهم، **«تقريباً صحيح» = خطأ.** سبع ركائز: (١) الحقيقة تُشتق لا تُدار · (٢) الثقة تُبنى بالشفافية · (٣) الامتثال جزء من المنتج لا إضافة · (٤) العزل مقدّس · (٥) نلتقي العميل حيث هو (الترحيل) · (٦) العمق قبل الاتساع · (٧) نعرف ما لسنا إياه.

## ثانياً: مبادئ المنتج
| # | المبدأ | اختبار القبول/الرفض |
|---|---|---|
| م-١ | قدسية سجل الحقيقة | تُرفض أي ميزة تسمح بتعديل يدوي لرقم مشتق أو تكسر ثبات العقد/التدقيق. |
| م-٢ | الامتثال أولاً | كل ميزة مالية تُسأل عن أثرها على ZATCA/Ejar/الفوترة؛ لا مستند مالي بلا ترقيم وطباعة ودفاع. |
| م-٣ | العزل غير قابل للتفاوض | تُرفض ميزة لا يمكن إثبات عزلها لكل مستأجر؛ كل سطح قراءة جديد محكوم بالهوية/الصلاحية. |
| م-٤ | الشفافية المُنطَّقة | كل طرف يرى بياناته فقط بالحدّ الأدنى؛ لا كشف زائد ولا تجميع عابر للحدود. |
| م-٥ | عربي-أولاً، سعودي-أصيل | RTL، تفقيط، هجري عند اللزوم؛ الإنجليزية ثانوية. |
| م-٦ | الترحيل بضغطة | كل كيان أساسي قابل للاستيراد الجماعي مع تحقّق ومعاينة وتراجع. |
| م-٧ | العمق قبل الاتساع | توسّع في مجاورة يؤجَّل ما دام الجوهر (SoR + الامتثال + التحصيل) غير مكتمل. |
| م-٨ | الخدمة الذاتية تقلّل التشغيل | تُفضّل الميزة التي تُغني عن اتصال بالمكتب. |
| م-٩ | كل إجراء قابل للتراجع أو مُدقَّق | لا حذف صلب؛ الإجراءات الحسّاسة تُسجَّل. |
| م-١٠ | لا ميزة تخمينية (YAGNI) | كل ميزة لها مستخدم حقيقي اليوم أو تُؤجَّل. |
| م-١١ | ندمج ولا نبني ما ليس جوهرنا | المدفوعات/الرسائل/ZATCA-API = تكامل لا بناء داخلي. |
| م-١٢ | القيمة تُحصَّل لا تُهدَر | كل ميزة كبرى تُسأل: هل تخدم نموذج الدخل أم تزيد كلفة بلا مقابل؟ |

## ثالثاً: مبادئ اتخاذ القرار
**بوابة الميزة (سبعة أسئلة):** الهوية · العزل/التدقيق · الامتثال · الشريحة · القيمة · بناء/تكامل/تأجيل · التكلفة الحقيقية. أي «لا» على (الهوية/العزل/الامتثال) = رفض فوري.
**أولوية التعارض:** الأمان ← صحّة/تدقيق ← امتثال ← تجربة ← سرعة/جَمال.
**هرمية البناء/التكامل/التأجيل:** نبني ما يمسّ سجل الحقيقة؛ ندمج المدفوعات/الرسائل/ZATCA/Ejar/المراقبة؛ نؤجّل الشقق المفروشة/الجوال/AI حتى يثبت الأساس التجاري.
**الوثيقة الواحدة:** أي قرار يغيّر السلوك يُسجَّل في الميثاق و`CHANGELOG`؛ لا ميزة تخالف الميثاق دون تعديله أولاً.
